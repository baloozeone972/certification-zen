# Gradle Professional

## 📋 Description
La certification Gradle Professional est une attestation de maîtrise des outils et techniques de build automation et gestion de projet avec Gradle. Cette certification est essentielle pour les développeurs et les chefs de projet qui souhaitent optimiser leur processus de développement et améliorer la qualité des projets logiciels.

## 🏛️ Organisme délivrant
**Gradle Inc.** - Compagnie de développement logiciel spécialisée dans la création d'outils pour l'automatisation du build et de gestion des projets.

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Gradle Build Script | 60 |
| Task Configuration | 50 |
| Dependency Management | 50 |
| Multi-project Builds | 50 |
| Plugins | 40 |

### Compétences évaluées
- Écrire et configurer des scripts Gradle
- Gérer les dépendances des projets
- Configurer et exécuter des tâches de build
- Gérer les builds multi-projets
- Utiliser et créer des plugins Gradle
