# Oracle Certified Professional Java 21 (OCP)

## 📋 Description
La certification OCP (Oracle Certified Professional Java 21) est une certification **pratique** qui atteste des compétences avancées en programmation Java. Elle couvre les dernières fonctionnalités de Java 21, notamment les virtual threads, le pattern matching avancé et d'autres améliorations. Cette certification est essentielle pour les développeurs souhaitant maîtriser la dernière version LTS de Java et être prêts à répondre aux défis du développement moderne.

## 🏛️ Organisme délivrant
**Oracle Corporation** - Leader mondial dans le domaine des technologies informatiques, Oracle propose cette certification pour vérifier les compétences en Java.

## 📅 Validité
- **Valable 3 ans**
- Renouvellement par examen ou certification supérieure (OCP Java 21)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Virtual Threads | 60 |
| Pattern Matching avancé | 70 |

### Compétences évaluées
- Utiliser les virtual threads pour améliorer la performance des applications Java
- Implémenter le pattern matching avancé pour une meilleure lisibilité et sécurité du code
...
