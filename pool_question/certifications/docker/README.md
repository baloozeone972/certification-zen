# Docker Certified Associate (DCA)

## 📋 Description

La certification Docker Certified Associate valide les compétences en conteneurisation avec Docker. Elle couvre la
création d'images, la gestion des conteneurs, les réseaux, les volumes, et l'orchestration avec Swarm.

## 🏛️ Organisme délivrant

**Docker, Inc.** - Créateur de la technologie de conteneurisation

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen

## 📚 Programme détaillé

### Thèmes principaux

| Thème                 | Pourcentage |
|-----------------------|-------------|
| Docker Basics         | 20%         |
| Images et Dockerfiles | 25%         |
| Networking Docker     | 15%         |
| Volumes et Stockage   | 15%         |
| Docker Compose        | 15%         |
| Docker Swarm          | 10%         |

### Compétences évaluées

- Créer et gérer des conteneurs Docker
- Écrire des Dockerfiles optimisés
- Configurer les réseaux (bridge, overlay, host)
- Gérer les volumes pour la persistance
- Orchestrer avec Docker Compose
- Déployer des services sur Docker Swarm
