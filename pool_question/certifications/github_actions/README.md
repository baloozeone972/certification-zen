## 📋 Description
La certification **GitHub Actions Professional** valide l'expertise technique des ingénieurs DevOps, SRE et développeurs dans l'automatisation des pipelines CI/CD directement sur GitHub. Elle couvre l'architecture YAML, l'orchestration des jobs, la gestion sécurisée des secrets, l'optimisation des runners et les stratégies de déploiement continu.

Conçue pour les environnements modernes, cette certification s'appuie sur les spécifications officielles de GitHub et intègre les dernières évolutions de la plateforme (runners Linux/Windows/macOS, OIDC, cache avancé, actions composées). Elle garantit au candidat une capacité opérationnelle immédiate à industrialiser, sécuriser et monitorer des workflows en production.

## 🏛️ Organisme délivrant
**GitHub (Microsoft Corporation)**. GitHub propose cette certification via son programme de validation des compétences techniques, reconnue par l'ensemble de l'écosystème open-source et enterprise. Le programme s'aligne sur les standards Cloud Native et les pratiques recommandées par la documentation officielle `docs.github.com/en/actions`.

## 📅 Validité
La certification est valable **2 ans**. Le renouvellement s'effectue via un examen de mise à jour (60 questions, 75% requis) couvrant les nouvelles fonctionnalités majeures de l'année écoulée. Un badge numérique vérifiable via OpenBadges est émis.
