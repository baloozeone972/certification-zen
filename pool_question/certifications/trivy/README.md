## 📋 Description
La certification **Trivy Security Professional** atteste de la capacité à sécuriser l'ensemble de la supply chain logicielle grâce à l'outil open-source Trivy d'Aqua Security. Elle couvre l'analyse de vulnérabilités OS et applicatives, la conformité Infrastructure as Code (Terraform, K8s, Dockerfile), et la génération automatisée de SBOM (CycloneDX, SPDX).

Ce programme est conçu pour les équipes DevSecOps, les architectes cloud et les ingénieurs sécurité souhaitant industrialiser la détection des risques dès la phase de build. Les candidats doivent maîtriser l'architecture client/serveur, les bases de données de vulnérabilités, l'écriture de règles OPA et l'intégration dans GitHub Actions, GitLab CI ou Jenkins.

## 🏛️ Organisme délivrant
**Aqua Security** – Leader mondial en sécurité cloud-native. Aqua maintient activement le moteur open-source Trivy, intégré dans les écosystèmes Kubernetes, CNCF et les principales plateformes de registre (Harbor, ECR, GCR, ACR).

## 📅 Validité
**24 mois**. Le renouvellement s'effectue via un examen de mise à niveau (delta) ou la présentation de 40 crédits de formation continue en sécurité cloud-native. Aqua publie des errata semestriels pour suivre les évolutions majeures du moteur.
