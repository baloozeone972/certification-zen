# Oracle Certified Professional Java 11 (1Z0-819)

## 📋 Description
La certification Oracle Certified Professional Java 11 (OCPJP) est une certification **pratique** qui atteste des compétences avancées en programmation Java et de la maîtrise des API modernes. Elle vise les développeurs qui souhaitent prouver leur expertise dans divers aspects de Java 11, y compris les modules, la programmation concurrente et l'utilisation des nouvelles fonctionnalités introduites dans les versions 9 à 11.

## 🏛️ Organisme délivrant
**Oracle Corporation** - Leader mondial dans le domaine du développement logiciel et des technologies informatiques

## 📅 Validité
- **Valable 3 ans**
- Renouvellement par examen

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Java Modules (JPMS) | 50 |
| Collections et Génériques avancés | 60 |
| Lambda et Streams API | 70 |
| Concurrence | 60 |
| NIO.2 et I/O | 50 |
| JDBC et Bases de données | 50 |
| Annotations et Reflection | 40 |
| Nouveautés Java 9-11 | 70 |

### Compétences évaluées
- Maîtriser les modules Java (JPMS)
- Utiliser efficacement les collections et génériques
- Manipuler les expressions lambda et l'API Streams
- Gérer la programmation concurrente
- Utiliser les fonctionnalités de NIO.2 et I/O
- Interagir avec des bases de données via JDBC
- Utiliser les annotations et la réflexion Java
- Exploiter les nouvelles fonctionnalités de Java 9 à 11
