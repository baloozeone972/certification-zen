# Oracle Certified Associate Java 8 (1Z0-808)

## 📋 Description
La certification Oracle Certified Associate Java 8 (1Z0-808) est une certification **fondamentale** pour les développeurs Java. Elle vise à évaluer les compétences de base en programmation Java et couvre une gamme complète des concepts clés, des types de données aux structures de contrôle, en passant par la programmation orientée objet et les API de base. Cette certification est essentielle pour ceux qui souhaitent démarrer une carrière en développement Java ou augmenter leurs compétences existantes.

## 🏛️ Organisme délivrant
**Oracle Corporation** - L'un des leaders mondiaux dans le domaine du développement logiciel et des technologies informatiques.

## 📅 Validité
- **Valable 3 ans**
- Renouvellement par examen ou certification supérieure (Oracle Certified Professional Java SE 8 Developer)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Fondamentaux Java | 60 |
| Types de données et opérateurs | 50 |
| Structures de contrôle | 50 |
| POO - Héritage et polymorphisme | 70 |
| Exceptions | 40 |
| API de base (String, ArrayList) | 60 |
| Lambda et Streams | 40 |
| Date et Time API | 30 |

### Compétences évaluées
- Comprendre les fondamentaux de la programmation Java.
- Manipuler des types de données et des opérateurs en Java.
- Utiliser les structures de contrôle pour contrôler le flux d'exécution des programmes.
- Implémenter la programmation orientée objet en Java, notamment l'héritage et le polymorphisme.
- Gérer les exceptions pour améliorer la robustesse des applications.
- Utiliser les API de base comme String et ArrayList.
- Écrire du code Java en utilisant des expressions lambda et les Streams.
- Manipuler les dates et les heures avec la Date et Time API.
