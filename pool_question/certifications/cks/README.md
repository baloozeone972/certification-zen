# Certified Kubernetes Security Specialist (CKS)

## 📋 Description

Certification CKS - Sécurité avancée des clusters Kubernetes, hardening, runtime security, supply chain, et compliance.

## 🏛️ Organisme délivrant

**CNCF (Cloud Native Computing Foundation)** - Linux Foundation

## 📅 Validité

- **Valable 2 ans**
- Nécessite CKA actif comme prérequis

## 📚 Programme

- Cluster Hardening
- System Hardening
- Supply Chain Security
- Monitoring & Logging
- Runtime Security
- Compliance

## 💼 Reconnaissance

⭐⭐⭐⭐⭐ (Très élevé - Certification sécurité K8s la plus reconnue)

## 💰 Salaires (France)

- Security Engineer : 70 000 - 95 000 €
- Platform Security : 85 000 - 120 000 €
