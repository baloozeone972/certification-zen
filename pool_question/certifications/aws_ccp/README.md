# AWS Certified Cloud Practitioner

## 📋 Description

Certification AWS de base - Fondamentaux du cloud AWS, services principaux, sécurité, tarification et conformité.

## 🏛️ Organisme délivrant

**Amazon Web Services (AWS)**

## 📅 Validité

- **Valable 3 ans**
- Renouvellement par examen

## 📚 Programme

- Cloud Concepts (IaaS, PaaS, SaaS)
- AWS Core Services (EC2, S3, VPC)
- Security & Compliance (IAM, Shared Responsibility)
- Billing & Pricing (TCO, Calculators)
- Architecture & Design
