# Prometheus Associate (CNCF)

## 📋 Description

La certification Prometheus Associate valide les compétences en monitoring et observabilité avec Prometheus, incluant la
collecte de métriques, PromQL, et l'alerting avec Alertmanager.

## 🏛️ Organisme délivrant

**Cloud Native Computing Foundation (CNCF)** - Fondation open source sous la Linux Foundation

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen

## 📚 Programme détaillé

### Thèmes principaux

| Thème                   | Pourcentage |
|-------------------------|-------------|
| Prometheus Architecture | 20%         |
| Metrics and Exporters   | 25%         |
| PromQL                  | 30%         |
| Alerting (Alertmanager) | 15%         |
| Grafana Integration     | 10%         |

### Compétences évaluées

- Déployer et configurer Prometheus
- Collecter des métriques via exporters
- Écrire des requêtes PromQL
- Configurer des règles d'alerting
- Intégrer Prometheus avec Grafana
- Comprendre l'architecture pull-based
