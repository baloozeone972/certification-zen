# GitLab Associate

## 📋 Description

La certification GitLab Associate valide les compétences dans l'utilisation de GitLab pour le versioning, la CI/CD, et
l'écosystème DevOps intégré.

## 🏛️ Organisme délivrant

**GitLab Inc.** - Plateforme DevOps complète

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen

## 📚 Programme détaillé

### Thèmes principaux

| Thème           | Pourcentage |
|-----------------|-------------|
| GitLab CI/CD    | 35%         |
| GitLab Runner   | 20%         |
| Git Basics      | 20%         |
| GitLab Security | 15%         |
| GitLab Packages | 10%         |

### Compétences évaluées

- Configurer des pipelines CI/CD (.gitlab-ci.yml)
- Gérer les runners (shared, specific, group)
- Maîtriser Git (branches, merge requests, rebase)
- Utiliser les fonctionnalités de sécurité (SAST, DAST)
- Gérer les packages et registries
