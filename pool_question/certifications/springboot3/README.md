# Spring Boot 3 Professional

## 📋 Description
La certification Spring Boot 3 Professional est une certification **pratique** qui vise à démontrer les compétences en développement d'applications microservices avec Spring Boot. Cette certification couvre l'auto-configuration, les starters et les actuators de Spring Boot, permettant aux développeurs de créer des applications robustes et évolutives.

## 🏛️ Organisme délivrant
**Pivotal Software Inc.** - Entreprise leader dans les solutions de développement d'applications basées sur Java

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (Spring Boot 3 Developer)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Spring Boot Auto-configuration | 60 |
| Spring Boot Starters | 50 |
| Spring Boot Actuator | 50 |
| Spring Boot Testing | 50 |
| Spring Boot DevTools | 40 |
| Spring Boot with Docker | 50 |

### Compétences évaluées
- Configurer et personnaliser les applications Spring Boot avec l'auto-configuration
- Utiliser les starters pour accélérer le développement des applications
- Instrumenter et surveiller les applications avec Spring Boot Actuator
- Écrire des tests unitaires et d'intégration pour les applications Spring Boot
- Utiliser Spring Boot DevTools pour améliorer l'expérience de développement
- Déployer des applications Spring Boot dans un conteneur Docker
