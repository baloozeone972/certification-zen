# Maven Professional (Apache)

## 📋 Description
La certification **Maven Professional** est une attestation de maîtrise des outils et des processus de gestion de projets Java avec Apache Maven. Cette certification démontre votre capacité à configurer et à gérer efficacement des projets Java en utilisant Maven, ainsi que votre connaissance approfondie de son écosystème. Elle est particulièrement appréciée dans les environnements de développement Java où la gestion des dépendances et le build automatisé sont essentiels.

## 🏛️ Organisme délivrant
**Apache Software Foundation (ASF)** - Fondation open source qui développe et maintient Apache Maven.

## 📅 Validité
- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (Maven Expert)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| POM Configuration | 60 |
| Build Lifecycle | 50 |
| Dependencies | 50 |
| Plugins | 50 |
| Multi-module Projects | 40 |

### Compétences évaluées
- Configurer et personnaliser les fichiers POM (Project Object Model)
- Gérer le cycle de build avec Maven
- Gérer les dépendances des projets Java
- Utiliser et configurer des plugins Maven
- Gérer des projets multi-modules avec Maven
