# DevSecOps Engineer (Linux Foundation)

## 📋 Description

Certification DevSecOps - Intégration de la sécurité dans le cycle DevOps (SAST, DAST, container security, compliance as
code).

## 🏛️ Organisme délivrant

**Linux Foundation**

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen

## 📚 Programme

- SAST & DAST
- Container Security
- Infrastructure Security
- Secrets Management
- Compliance as Code
- Security Pipelines

## 💼 Reconnaissance

⭐⭐⭐⭐ (Élevé - Certification sécurité DevOps la plus complète)

## 💰 Salaires (France)

- DevSecOps Engineer : 65 000 - 95 000 €
- Security Engineer : 70 000 - 100 000 €
