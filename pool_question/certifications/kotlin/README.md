# Kotlin Developer (JetBrains)

## 📋 Description
La certification Kotlin Developer est une attestation de maîtrise du langage de programmation Kotlin, développé par JetBrains. Cette certification vise à évaluer les compétences en développement moderne avec Kotlin, couvrant des aspects tels que la syntaxe Kotlin, la gestion de nullité, les coroutines, la programmation fonctionnelle et l'orienté objet. Elle est particulièrement appréciée pour son utilisation dans les environnements JVM, Android et multiplateformes.

## 🏛️ Organisme délivrant
**JetBrains** - Compagnie de développement logiciel spécialisée dans l'offre d'outils et technologies pour le développement de logiciels.

## 📅 Validité
- **Valable 3 ans**
- Renouvellement par examen ou certification supérieure (Kotlin Developer Expert)

## 📚 Programme détaillé
### Thèmes principaux
| Thème | Questions |
|---|---|
| Kotlin Syntax | 60 |
| Null Safety | 50 |
| Coroutines | 70 |
| Functional Programming | 60 |
| Object-oriented | 60 |

### Compétences évaluées
- Maîtriser la syntaxe Kotlin
- Gérer la gestion de nullité efficacement
- Utiliser les coroutines pour des tâches asynchrones et non bloquantes
- Appliquer la programmation fonctionnelle dans le développement
- Concevoir et implémenter des classes et des interfaces en orienté objet
