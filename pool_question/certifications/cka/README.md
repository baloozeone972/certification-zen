# Certified Kubernetes Administrator (CKA)

## 📋 Description

La certification CKA (Certified Kubernetes Administrator) est une certification **pratique** qui valide les compétences
d'administration de clusters Kubernetes. L'examen se fait dans un environnement terminal réel.

## 🏛️ Organisme délivrant

**Cloud Native Computing Foundation (CNCF)** - Fondation open source sous la Linux Foundation

## 📅 Validité

- **Valable 2 ans**
- Renouvellement par examen ou certification supérieure (CKS)

## 📚 Programme détaillé

### Thèmes principaux

| Thème                  | Pourcentage |
|------------------------|-------------|
| Cluster Architecture   | 25%         |
| Workloads & Scheduling | 20%         |
| Services & Networking  | 20%         |
| Storage                | 10%         |
| Troubleshooting        | 15%         |
| Security               | 10%         |

### Compétences évaluées (pratiques)

- Installer et configurer un cluster Kubernetes
- Gérer les workloads (pods, deployments, services)
- Configurer le networking (Ingress, Network Policies)
- Gérer le stockage (PersistentVolumes, StorageClasses)
- Diagnostiquer et résoudre les problèmes
- Sécuriser le cluster (RBAC, SecurityContexts)

## 🎯 Objectif

Maîtriser l'administration de clusters Kubernetes pour des environnements de production.

## 🚀 Débouchés

- Kubernetes Administrator
- DevOps Engineer (spécialisé conteneurs)
- Platform Engineer
- SRE (Site Reliability Engineer)

## 📊 Statistiques

- **Taux de réussite** : ~60%
- **Temps de préparation recommandé** : 100-150 heures
- **Format examen** : Pratique (terminal), 17 questions, 120 minutes
